# Understanding-Financial-Market-Dynamics-through-Linear-Algebra-
The main objective of the project is to use Python, Linear Algebra, and  real-world data to analyze how shocks in the market affect the stability of  these financial institutions in India, such as SBI, ICICI, HDFC, and Axis Bank.  We have to calculate Eigenvalues , Eigenvectors, etc in the project for the  analysis.
